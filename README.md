1. Global Intelligence
The app uses the ArcGIS engine to understand the world. Unlike basic search bars, this can resolve landmarks, specific addresses, or entire cities across the globe (e.g., searching "Eiffel Tower" to "Big Ben"). It then uses geodesic mathematics to calculate the most accurate "great-circle" distance between those coordinates.
2. The GSF Sustainability Engine (The "Brain")
This is the heart of the project. It follows the Green Software Foundation (GSF) standards to calculate a Software Carbon Intensity (SCI) score.

Operational Emissions: The carbon produced by the fuel or energy used during the trip.

Embodied Emissions: The "hidden" carbon cost of the hardware (your phone or computer) used to run the routing software.
